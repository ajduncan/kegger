# kegger #

Woo, kegger!

## Installing ##

	OSX/Linux:

	$ virtualenv -p python3 .env
	$ source .env/bin/activate
	$ (.env) pip install -r requirements.txt

## Running ##

	$ (.env) python kegger/app.py

	View app @ http://localhost:5000/
